    1 What is the difference between var, let, and const?
    Ans:
    var :var is function-scoped and can be re-declared. We avoid it in modern JS.
    let : let is block-scoped and can be reassigned.
    const : const is block-scoped and cannot be reassigned.
    We use Default to const, use let only when we need to reassign.

    2 What is the spread operator (...)?
    Ans:
    It "spreads" an array or object into individual items. Useful for copying or merging without mutating the original.

    3 What is the difference between map(), filter(), and forEach()?
    Ans:
    map() — transforms every item and returns a NEW array of the same length.
    filter() — keeps only items that pass a condition and returns a NEW array.
    forEach() — just runs code for each item, returns nothing (undefined).

    4 What is an arrow function?
    Ans:
    An arrow function is a shorter syntax for writing functions in JavaScript. It was introduced in ES6 (ECMAScript 2015) and makes code more concise and readable.

    5 What are template literals?
    Ans:
    Template literals are a modern way to work with strings in JavaScript. They allow you to embed variables and expressions directly inside a string.
    Template literals use backticks (`) instead of single or double quotes.
